Contact Form

  • Make sure your website is on server, php not working localy. so first you need to put the website on server for make contact form working
  • Make sure all the path you given into html/contact-form/js/smart-form.js is correct
  • Open html/contact-form/php/settings.php and add your email to receiver email address - $receiver_email = "example@domain.com";
  • After making the changes - contact form will working fine as work on live demo

Sass uses

Versa is built using sass and gulp. with sass you can easily manage your css customization


If you are not familiar with sass, then you can use the style.css file directly for customizations
custom css file - html/css/style.css

If you are using sass, you have a good news because Versa is built using sass.
Sass is compiled by GULP

How to make working sass with gulp

We're say special thank to css tricks who help me out for working with gulp you can easily learn how to use gulp and how to compile file. click on this link and get quick start with gulp - Gulp for beginners

All plugins jquery files compile into one file - js/bundle/bundle.js
